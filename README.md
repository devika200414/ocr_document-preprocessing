
# OCR Document Processing and Data Extraction

This project implements an automated system to extract key information from scanned corporate documents (such as invoices) using OCR and basic data processing techniques.

## Project Objective
To automate document processing and data extraction from unstructured documents using OCR and Computer Vision.

## Technologies Used
- Python  
- OpenCV  
- Tesseract OCR  
- Pandas  
- NumPy  
- Google Colab (for development)

## Project Structure


## Features
- Upload or provide scanned document (image or PDF)  
- Extract text using OCR  
- Identify key fields such as:
  - Invoice Number  
  - Date  
  - Amount  
- Display structured output
